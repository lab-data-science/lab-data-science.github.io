Data Science Lab

IMT School for Advanced Studies

Instructor: Falco J. Bargagli Stoffi


Overview:

This course will provide an introduction to R for data science and policy evaluation. R is a statistical software that is becoming more important and more widely used both in Academia and in the private sector. This is due to the fact that this programming language is, among the other things, (i) flexible, (ii) tailored for data analysis and machine learning and (iii) open source. During each class, a series of applications useful to develop autonomous research projects will be presented. At the end of the course students will be able to perform from scratch a data analysis in R.

Format:

Interactive approach: students will be asked to perform simple analysis during the lectures after the main concepts are introduced. Remote lectures available at this link. The exam will consist in an individual project/presentation by each student using one or multiple tools introduced during the course.

Prerequisites:

Students are asked to install R and R Studio before the first lecture. Here a breif guide on how to install both.
